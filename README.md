# DSA Concept First 🧠

A day-by-day journey through Data Structures & Algorithms — starting from concepts, not shortcuts.

---

## 🎯 Approach

Instead of jumping directly into coding questions, I follow a structured path:

1. **Learn the concept** — Understand how the data structure works internally, its memory behavior and use-cases
2. **Implement from scratch** — Write code for basic operations (like CRUD) without shortcuts or built-in methods
3. **Build intuition** — Understand when and why to use each data structure
4. **Then move to problem solving** — Pattern-based questions (Sliding Window, Recursion, Graphs, etc.)
