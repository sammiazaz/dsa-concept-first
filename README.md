# 📘 DSA Concept-First Journey

## 🚀 Roadmap of My Learning

This repository documents my journey of learning Data Structures and Algorithms (DSA) with a clear focus on understanding concepts deeply before solving problems.

---

## 🎯 Approach

Instead of jumping directly into coding questions, I am following a structured path:

1. **Learn the concept**

   * Understand how the data structure works internally
   * Learn memory behavior and use-cases

2. **Implement from scratch**

   * Write code for basic operations (like CRUD)
   * Avoid shortcuts or built-in methods initially

3. **Build intuition**

   * Understand when and why to use each data structure

4. **Then move to problem solving**

   * Pattern-based questions (Sliding Window, Recursion, Graphs, etc.)
